Tag 2: Variablen

Heute wirst du lernen, wie man Variablen und Datentypen in Python verwendet.

Was sind Variablen?

Variablen sind "Behälter" für Daten. Du kannst ihnen Werte zuweisen, die du später in deinem Programm verwenden kannst. Zum Beispiel:

name = "Max"

Hier haben wir eine Variable `name`, der der Wert `"Max"` zugewiesen wurde.

Datentypen in Python

Python kennt verschiedene Datentypen, die unterschiedliche Arten von Daten speichern. Heute wollen wir uns mit Strings und Integer beschäftigen:

  • String (Text): Wird in Anführungszeichen gesetzt, z.B. `"Hallo"`
  • Integer (Ganzzahl): Ganze Zahlen, z.B. 5

Öffne die Entwicklungsumgebung

name = "Max"
alter = 25


print(name)
print(alter)

In diesem Beispiel haben wir zwei Variablen: `name` und `alter`. Sie haben verschiedene Datentypen, aber wir können sie mit der `print()`-Funktion auf dem Bildschirm anzeigen lassen. Dafür können wir einfach den Variablen Name in die Klammer setzen.
