Tag 10: Funktionen in Python

Heute lernst du, wie man Funktionen in Python erstellt und verwendet. Funktionen sind sehr nützlich, um wiederverwendbaren Code zu schreiben.

Was sind Funktionen?

Funktionen ermöglichen es dir, einen Codeblock zu erstellen, den du immer wieder aufrufen kannst. Du definierst eine Funktion mit dem Schlüsselwort `def` und kannst ihr beliebig viele Eingabewerte (Parameter) übergeben. Hier ist ein einfaches Beispiel:

def begruessung(name):
    print("Hallo, " + name + "!")

In diesem Beispiel haben wir eine Funktion `begruessung()`, die einen Namen entgegennimmt und eine Begrüßung ausgibt.

Funktion aufrufen

Um die Funktion zu verwenden, rufst du sie mit einem Wert für den Parameter auf. Zum Beispiel:

begruessung("Max")

Dies gibt die Nachricht "Hallo, Max!" aus.

Schritt 1: Deine eigene Funktion erstellen

Öffne die Entwicklungsumgebung und schreibe folgenden Code:

def addiere(x, y):
    return x + y

ergebnis = addiere(5, 3)
print(ergebnis)

In diesem Beispiel haben wir eine Funktion `addiere()`, die zwei Zahlen addiert und das Ergebnis zurückgibt. Wir rufen die Funktion mit den Werten 5 und 3 auf und speichern das Ergebnis in der Variablen `ergebnis`.

Schritt 2: Funktion ausführen

Führ die Datei aus, um das Ergebnis der Addition zu sehen.
