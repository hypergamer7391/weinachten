Tag 7: Listen

Heute wollen wir uns mit Listen beschäftigen.

Was sind Listen?

Listen sind ein Variablen Typ wie ein String. Der Unterschied ist das man mehrere Zahlen oder Texte speichern kann. Wichtig zu beachten: Es kann immer nur ein Variablen Typ gespeichert werden. 

zahlen = [5, 12, 7]

In diesem Beispiel werden die Zahlen 5, 12 und 7 in der Liste gespeichert.

Einzelne Elemente aus einer Liste ausgeben

Kopier folgenden Code in den Code Editor:

zahlen = [5, 8, 10, 18]
print(zahlen[1])

Führ die Datei nun aus aus. Du wirst sehen, Eine Zahl aus der Liste ausgegeben wird. Ändere nun die Zahl in der eckigen Klammer [] hinter zahlen und schau was passiert. Ersetz die Zahl auch durch 0

Diese Zahl heißt Index. Der Index gibt an welche Stelle der Liste ausgegeben oder weiter genutzt werden soll. Wie du nach dem Ausprobieren vermutlich schon gemerkt hast wird beim Index 1 nicht 5 sondern 8 ausgegeben dies liegt daran das die erste Stelle der Liste nicht mit dem Index 1 sondern 0 beschrieben wird. Dadurch muss man immer darauf achten das wenn man eine Stelle ausgeben möchte man nicht die Stelle als Index angibt sondern die Stelle-1.
