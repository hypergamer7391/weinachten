Tag 16: Klassen – Erstelle eine Klasse Weihnachtsbaum mit einem Attribut Höhe

Heute lernst du, wie du mit Python Klassen erstellst. Klassen sind ein zentraler Bestandteil der objektorientierten Programmierung und helfen, Strukturen zu organisieren.

Programm: Klasse Weihnachtsbaum

class Weihnachtsbaum:
    def __init__(self, hoehe):
        self.hoehe = hoehe

    def anzeigen(self):
        print(f"Dieser Weihnachtsbaum ist {self.hoehe} Meter hoch.")

baum = Weihnachtsbaum(3)
baum.anzeigen()

Erklärung:

  1. Klasse: Eine Klasse ist eine Vorlage für Objekte, die Attribute und Methoden enthalten kann.
  2. Init-Methode: Die Methode __init__ wird aufgerufen, wenn ein Objekt erstellt wird. Sie dient zur Initialisierung von Attributen.
  3. Methode: Die Methode anzeigen gibt die Höhe des Weihnachtsbaums aus.

Challenge des Tages:

Erweitere die Klasse so, dass sie auch eine Farbe enthält. Füge eine Methode hinzu, die die Höhe und Farbe des Weihnachtsbaums anzeigt.
