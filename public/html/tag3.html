Tag 3: Eingaben

Heute wirst du lernen, wie man Eingaben von Nutzern in Python nutzen kann.

Was sind Eingaben?

Eingaben sind Interaktionen mit dem Nutzer. Ein Beispiel für eine Eingabe ist:

Was ist dein Name? Max

Hier haben wir eine Frage die den Nutzer dazu auffordert seinen Namen einzugeben. Der Name den er eingibt ist nun die Eingabe

Einagben in Python

Es gibt eine einfache möglichkeit um Eingaben vom Nutzer zu bekommen::

  • Die `input()`-Funktion: `input()`-Funktion funktioniert ähnlich wie die Print Funktion nur das der Nutzer danach etwas eingeben kann.
  • Es wird zuerst eine voher festgelegte Frage ausgeben auf die der Nutzer dann etwas in die Konsole eingeben kann. Die Eingabe des Nutzers wird dann in einer Variable gespeichert.

Code

name = input("Wie heißt du? ")

In diesem Beispiel haben wir die Variable: `name` . Sie wird erst durch die Eingabe festgelegt. Also erst wenn man seinen Namen eingibt bekommt die Variable einen Wert.

Kopier das Beispiel in die Entwicklungsumgebung und führ den Code aus.

Du solltest nun in der Konsole gefragt werden wie du heißt und wenn du deinen Namen eingibst sollte nichts passieren.

name = input("Wie heißt du? ")

print("Dein Name ist: "+name)

Wir haben nun noch eine `print()`-Funktion hinzugefügt um den Namen auszugeben.

Ändere nun auch deinen Code und führ ihn aus, die Ausgabe sollte nun so aussehen:

Wie heißt du?
