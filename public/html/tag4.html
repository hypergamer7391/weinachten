Tag 4: Bedingungen und if-Anweisungen

Heute lernst du, wie du Bedingungen mit `if`-Anweisungen in Python verwendest. Diese helfen dir, Entscheidungen zu treffen, basierend auf bestimmten Kriterien.

Was sind Bedingungen?

Mit Bedingungen kannst du überprüfen, ob etwas wahr oder falsch ist, und je nach Ergebnis verschiedene Aktionen ausführen. Die grundlegendste Form einer Bedingung ist die `if`-Anweisung:

alter = 20
if alter >= 18:
    print("Du bist volljährig.")

In diesem Beispiel überprüft Python, ob der Wert in der Variablen `alter` größer oder gleich 18 ist. Wenn dies der Fall ist, wird die Nachricht "Du bist volljährig." ausgegeben.

Die `else`-Anweisung

Du kannst auch eine `else`-Anweisung hinzufügen, die ausgeführt wird, wenn die Bedingung nicht erfüllt ist:

alter = 16
if alter >= 18:
    print("Du bist volljährig.")
else:
    print("Du bist noch minderjährig.")

In diesem Fall wird die Nachricht "Du bist noch minderjährig." ausgegeben, weil die Bedingung nicht erfüllt ist.

Eine Bedingung erstellen

Schreibe folgenden Code in die Entwicklungsumgebung:

alter = 25
if alter >= 18:
    print("Du bist volljährig.")
else:
    print("Du bist noch minderjährig.")

Führ die Datei aus. Je nach Wert von `alter` wird eine der beiden Nachrichten angezeigt.
