Tag 5: Schleifen in Python

Einführung

Heute lernst du, wie Schleifen in Python verwendet werden können. Schleifen ermöglichen es, Codeblöcke mehrmals auszuführen, ohne sie wiederholt schreiben zu müssen.

1. Was sind Schleifen?

Eine Schleife ist ein wiederholter Ablauf von Code. In Python gibt es zwei Haupttypen von Schleifen:

  • for-Schleife: Wiederholt eine festgelegte Anzahl von Durchläufen.
  • while-Schleife: Wiederholt sich, solange eine Bedingung wahr (True) ist.

2. Die for-Schleife

Eine for-Schleife durchläuft eine Sequenz von Werten oder eine festgelegte Anzahl an Iterationen. Hier ist ein Beispiel:

for i in range(5):
    print("Frohe Weihnachten!")

Die Schleife läuft fünfmal, da range(5) die Zahlen von 0 bis 4 erzeugt. Jede Iteration führt dazu, dass die Nachricht "Frohe Weihnachten!" ausgegeben wird.

3. Die while-Schleife

Die while-Schleife wiederholt den Code, solange die Bedingung wahr (True) ist. Hier ist ein Beispiel:

count = 0
while count < 5:
    print("Frohe Weihnachten!")
    count += 1

In diesem Beispiel startet die Variable count bei 0. Solange count kleiner als 5 ist, wird die Nachricht "Frohe Weihnachten!" ausgegeben und count bei jedem Durchlauf um 1 erhöht. Die Schleife endet, sobald count den Wert 5 erreicht.

Wichtig! mit dem befehl break kannst du eine while-Schleife sofort abbrechen. Dass brauchst du in zukunftigen Aufgaben

4. Challenge des Tages

Nutze eine for-Schleife so, dass die Nachricht mit einer Nummerierung ausgegeben wird, z. B.:


1 Frohe Weihnachten!
2 Frohe Weihnachten!
3 Frohe Weihnachten!

Errinere dich daran wie man Variablen in eine print()-Funktion einsetzen kann(Tag 2). Wenn du Hilfe brauchst probier einmal diesen Code auszuführen:

for i in range(1,4):
    print(i)
