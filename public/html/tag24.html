Tag 24: Projekt – Mini-Adventskalender

Zum Abschluss erstellst du ein Programm, das für jeden Tag im Dezember bis Weihnachten eine Nachricht anzeigt.

Programm: Adventskalender

from datetime import datetime

nachrichten = {
    1: "Starte in die Weihnachtszeit!",
    2: "Zeit, Kekse zu backen!",
    # Füge für jeden Tag eine Nachricht hinzu
    24: "Frohe Weihnachten!"
}

heute = datetime.now().day

if heute in nachrichten:
    print(nachrichten[heute])
else:
    print("Noch keine Nachricht für heute.")

Erklärung:

  1. Dictionary: Die Nachrichten für jeden Tag werden in einem Dictionary gespeichert.
  2. Aktuelles Datum: Mit datetime.now() wird das aktuelle Datum abgerufen.
