Tag 1: Einführung in Python

Willkommen zu deinem ersten Tag des Python Adventskalenders! Heute lernst du, was Python ist, und schreibst dein erstes Programm.

Was ist Python?

Python ist eine der beliebtesten Programmiersprachen der Welt. Sie wird in vielen Bereichen eingesetzt, z.B. Webentwicklung, Datenanalyse und maschinelles Lernen. Python ist besonders für Anfänger geeignet, da sie eine einfache und gut lesbare Syntax hat.

Der erste Code

Jetzt können wir anfangen unser erstes Python-Programm zu schreiben und auszuführen. Folge dafür diesen Schritten:

  • Schreibe den folgenden Code in die Datei:
    • print("Hallo, Welt!")
  • Um das Programm auszuführen, musst du nun auf Run klicken.
  • Wenn alles richtig eingerichtet ist, sollte der Text „Hallo, Welt!“ im Terminal angezeigt werden.

Challenge des Tages:

Ändere den Text in der `print()`-Funktion und lasse es deinen Namen anzeigen, zum Beispiel:

print("Hallo, Max!")
