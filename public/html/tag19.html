Tag 19: Listen-Komprehension – Erstelle eine Liste von Zahlen

Heute lernst du, wie du Listen-Komprehension nutzt, um Listen in einer kompakten Form zu erstellen.

Programm: Zahlenliste erstellen

zahlen = [x for x in range(1, 11)]
print("Liste der Zahlen von 1 bis 10:", zahlen)

Erklärung:

  1. Listen-Komprehension: Mit Listen-Komprehension kannst du Listen mit einer einzigen Zeile Code erstellen.
  2. Range: Die Funktion range() erzeugt eine Sequenz von Zahlen, die durchlaufen wird.
